Entities and Symbols

To use some special symbols in html we need to use entities becaous html cannot understand them as character rather consider them as a part of element.

EXAMPLE:

1. for less than sign use : <

2. for grater than sign use : >

3. for hyphen than sign use : ‑

4. for no break line space use       this is after 5 spaces

5. for copyright symbol use : ©

6. for trademark symbol use : ™

7. for registred symbol use : ®

8. to use apostrophe : It's

8. to use double quotes: "this is inside quotes"

9.to use grave marks: à

10.to use acute marks: b́

11.to use circumflex sign : â

12.to use tilde sign: ̃x


Emojis

Emojis are not images they are charater with specific UNICODE

To use them we need to add meta charset="UTF-8 " with in head tag.
