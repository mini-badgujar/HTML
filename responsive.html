Responsive Webpages

there are diffrent ways to make a webpage responsive

  • Setting viewport
  • Setting diffrent element size and width such as images and text, according to screen size
  • Setting picture element to display multipal pictures based on screen size
  • Using media queries and resizing the browser window

Viewport setting in this webpage we can see the diffrence between a normal page and a veiwport set page.

setting this in meta: meta name="viewport" content="width=device-width, initial-scale=1.0"

Here the conent width is set as device width which means that the content size will change according to screen size
